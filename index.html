Common questions

Do I need coding experience?

No. If you can describe what you want, we’ll help Cursor and Claude build it. We scaffold everything so beginners can follow and advanced builders don't get bored.

Can I work on my own idea?

Yes, that’s the point. Bring your Figma file, existing repo, or rough sketch. We tailor support so you leave with something real.

What if I get stuck?

You get weekly calls and support between calls, so you’re not stuck for days when something breaks or the AI gives you confusing output.

How is this different from a video course?

This is not passive content. It’s a guided build program where we walk you through the full path from idea to deployed app, help you make product and technical decisions, and get you unstuck when AI, code, or tools break. You work on your own app, not generic lessons.

What tools do I need?

Node.js, Git, Cursor, Claude access, GitHub, and Vercel. We send a preflight checklist and help you get set up before the first build sprint.

Is there follow-up support?

Yes. You get lifetime access to the content and workflows, plus support during the program while you build your own app.

Do you offer financial aid?

This is a high-support program for people serious about learning to build real software with AI. Right now we are not offering financial aid.
